# Hyperbolic Geometry Android App

Интерактивное Android-приложение для визуализации геометрических объектов в модели Бельтрами-Клейна.

## Функциональность

- **Гиперболическая сфера** - интерактивная 3D-визуализация с управлением параметрами
- **Орисфера** - модель орисферы с геодезическими линиями
- **Эквидистантная поверхность** - визуализация эквидистантных поверхностей
- **Пучки прямых** - эллиптические, параболические и гиперболические пучки

## Требования

- Android 7.0 (API level 24) и выше
- OpenGL ES 3.0 (для 3D-графики)

## Технологии

- **Kotlin** - основной язык разработки
- **OpenGL ES 3.0** - 3D-визуализация
- **Jetpack Compose** - UI
- **MVVM** - архитектура приложения
